**# Portafolio de Proyectos — Desarrollo de Aplicaciones para Dispositivos Móviles**

• Jesús Antonio Arellano Zapata
• Desarrollo de Aplicaciones para Dispositivos Móviles
• Profesor: Jesús Salas Marín
• 8vo Semestre

• Este portafolio contiene las prácticas y actividades realizadas durante la unidad, enfocadas en el aprendizaje del lenguaje Dart y el desarrollo de aplicaciones móviles con Flutter. En él se incluyen ejercicios sobre funciones básicas, funciones tipo, funciones anónimas, clases, objetos, herencia, manejo de datos en formato JSON y la creación de una primera aplicación en Flutter. Cada práctica contiene una explicación breve del procedimiento realizado, evidencias del código desarrollado y una conclusión personal sobre los aprendizajes obtenidos.

**Tecnologías utilizadas**
• Dart
• AppSheet
• Flutter
• Visual Studio Code
• GitHub

• A través de estas actividades desarrollé competencias relacionadas con la programación estructurada y orientada a objetos. Aprendí a definir funciones, reutilizar código, pasar funciones como parámetros y utilizar funciones anónimas para resolver tareas específicas de manera más flexible. También reforcé el uso de variables, condicionales, ciclos, listas, métodos y estructuras de control.
